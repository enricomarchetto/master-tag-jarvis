# master-tag-jarvis

Guide e starter pack per il setup di un vault Jarvis (Obsidian + Claude Code + Antigravity). Materiale didattico del Master AI for Advertising di Talent Garden, edizione Spring 2026.

## Cosa contiene questo repo

- **Guide di setup** dettagliate per macOS e Windows, in Markdown (leggibili direttamente su GitHub) e PDF (per stampa o lettura offline)
- **Starter pack** (`.zip`) con la struttura iniziale del vault: skill di Jarvis, template di note, configurazione Obsidian base. Disponibile come [Release asset](../../releases/latest)

## A chi si rivolge

Studenti del Master ITAVPT — AI for Advertising di Talent Garden, edizione Spring 2026. Più in generale, chiunque voglia replicare lo stesso setup vault Jarvis per uso personale o didattico.

## Come iniziare

1. Scegli la guida giusta per il tuo sistema operativo:
   - **macOS** → [guides/mac.md](guides/mac.md) ([versione PDF](guides/mac.pdf))
   - **Windows** → [guides/windows.md](guides/windows.md) ([versione PDF](guides/windows.pdf))
2. Scarica lo starter pack dalla [pagina Releases](../../releases/latest) (file `vault-starter-master-tag.zip`)
3. Segui la guida passo per passo

Tempo stimato: circa un'ora, una volta sola.

## Stack

- **[Warp](https://www.warp.dev)** — terminale moderno
- **[Node.js + npm](https://nodejs.org)** — runtime per Claude Code
- **[Claude Code](https://claude.com/claude-code)** — AI agentica di Anthropic, qui chiamata "Jarvis"
- **[Obsidian](https://obsidian.md)** — database di note in markdown
- **[Antigravity](https://labs.google.com/antigravity)** — IDE Google basato su VSCode con AI integrata
- **[Google Drive](https://www.google.com/drive)** (opzionale) — sincronizzazione del vault tra device
